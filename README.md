# imgc

Cross-platform image viewer inspired by the now archived [https://github.com/xyb3rt/sxiv](sxiv).

Motivation: I wanted a nice image viewer like sxiv that 1) supports wayland and 2) supports windows. I used SDL3 as a platform layer. It's pretty usable as and has replaced other image viewers for me, but its still WIP and I plan to add more fancy features later.
